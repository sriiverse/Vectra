Technical Architecture Reference

1

CLIP Embedding

Encodes images and text into a aligned 512-dimension vector space. Combines intent linearly (Query = Image + 0.3 * Text).

2

pgvector HNSW

Performs hard SQL filtering (price, category, stock) and executes cosine distance sort on pgvector column with an HNSW index.

3

Cross-Encoder Rerank

Jointly reviews query terms against product names of top-50 results using MiniLM-L6 to capture fine-grained similarity.
